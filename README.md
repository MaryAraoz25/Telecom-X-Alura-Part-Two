# Telecom-X-Alura-Part-Two
Predicción de Cancelación (Churn)
Descripción:
En este proyecto se desarrollaron modelos de Machine Learning para predecir la cancelación de clientes en Telecom X.
Proceso:
- Limpieza y preparación de los datos.
- Codificación de variables categóricas (One-Hot Encoding).
- Análisis de correlación y exploración de variables.
- División de datos en entrenamiento y prueba.
- Normalización de variables para algunos modelos.
Modelos:
Se entrenaron dos modelos de clasificación:
- Regresión Logística
- Random Forest
Los modelos se evaluaron usando métricas como accuracy, precision, recall, F1-score y matriz de confusión.
